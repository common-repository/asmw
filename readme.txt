ASMW allows you to promote music from Amie Street on your WordPress site.

== Description ==

The main goal is to post music from <a href="http://amiestreet.com/?pytr=asmw">Amie Street</a> on your website to help promote your favorite artist.  This is done every Wednesday as to help break up the week and give people a varity of music to listen to.

Use the plugin anytime you wish to promote a band, not jsut on Wednesdays.

== Installation ==

To install ASMW just extract the ASMW.zip file into your WordPress plugins directory.  By default it sould be 'dir_of_wordpress/wp-content/plugins'.

Then you will need to login your admin panel, go to plugins, and activate the plugin.  Now you are ready to use ASMW.

== Frequently Asked Questions ==

= How does the plugin work =

The plugin works by injecting text into your post before it is placed into your wordpress database.

= Who made the player =

Lucas @ Amie Street developed the flash player. The player itself is hosted on <a href="http://amiestreet.com/?pytr=asmw">Amie Street</a>.

= How do I use v3.x =

Now even easier.  Just copy the link location in your browser of the song and paste it in brackets [].  You can look at http://amiest.musicwednesday.com/howto for more information

= What is the Uber Tracker =

The Uber Tracker is where all of the pingbacks/trackbacks go.  This is nothing more than a WordPress page that tracks when an ASMW post goes up.  This allow everyone a central place to go and look for music that has been posted by people that use ASMW plugin.  This will also help drive some traffic to your site. :)
